BreakScheduleApp – Functional Specifications

The BreakScheduleApp is an iOS application that simplifies creating and managing staggered break schedules. It provides managers and employees with clear, real-time information while supporting role-based permissions.

Purpose & Scope

Enable team leads and store management to plan and publish break schedules, approve swap requests, and communicate with employees. The app supports multiple user roles: Store Director, ETL, TL, and TM.

Core Features

  • User Authentication: Sign in with Apple or username/password.
  • Role-Based Access: Permissions vary by Store Director, ETL, TL, and TM roles.
  • Break Scheduling: Create, edit, and publish daily break schedules with PDF export.
  • Shift Swaps: Employees can propose swaps; managers can approve or deny them.
  • Messaging: In-app direct messages with push notifications.
  • Cloud Sync: All data stored in CloudKit for real-time updates across devices.

User Stories & Use Cases

  • As a Team Lead, I can create a schedule and notify employees instantly.
  • As an Employee, I can request a break swap and receive approval or denial in real time.
  • As a Store Director, I can view and manage all department schedules.
